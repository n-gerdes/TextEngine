
get_name(): Returns the name of a scene
get_children(): Returns an array containing the true names of all the entities present in the scene
entity_exists($name): Returns whether or not there exists an entity with the specified true name.
entity_exists_here($name): Returns whether or not this scene contains an entity with the specified true name.
get_global_value($variable_name): Returns the value of the specified global variable.
get_meta_value($variable_name): Returns the value of the specified meta variable.
set_value($variable_name, $value): Set the value of the specified variable
get_value($variable_name): Return the value of the specified variable.


EVENT FUNCTIONS

initialize(): called when the scene is first loaded into the scenario.
describe(): called when the perspective entity enters a scene, intended to be used for having it describe the scene.
on_enter_scene(entity_name): called when an entity enters this scene
before_turn(entity_name): called on within a scene before a given entity takes its turn.
